Upload the contents of this folder into the root of the GitHub repository.

Important:

- Do not upload this folder as a subfolder.
- Do not upload the ZIP file to GitHub Pages.
- Select all files and folders inside this folder and drag them into the repository root.
- Replace the existing `index.html`, `style.css`, `script.js`, `data.json`, `.nojekyll`, `assets/`, and `SEc File/` if GitHub asks.

- Opening the GitHub Pages URL first shows the password screen.
- Password: `LMdashboard2026!`
- In the dashboard, `LM Assist Trend View` shows `Daily`, `WoW`, and `MoM`.

If GitHub still shows the old dashboard:

- Open the URL with `?v=20260521` appended once.
- Hard-refresh the browser.
